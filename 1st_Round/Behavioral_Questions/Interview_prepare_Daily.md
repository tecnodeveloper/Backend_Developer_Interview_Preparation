    ## **Introduction to Programming**

### 1. JavaScript aur Python mein difference simple words mein batao.

JavaScript mainly web aur asynchronous applications ke liye use hoti hai, jabke Python simple syntax aur AI/data science/backend mein popular hai.

### 2. Asynchronous programming Node.js mein kyun important hai?

Node.js single-threaded hai, async programming multiple requests ko efficiently handle karti hai without blocking server.

### 3. REST API kya hota hai aur backend mein iska role kya hai?

REST API frontend aur backend ke darmiyan communication ka standard way hai using HTTP methods like GET, POST, PUT, DELETE.

### 4. Tumhari coding workflow aur version control kaise handle karte ho?

Feature branch create karta hoon, code locally test karta hoon, commits properly maintain karta hoon, then pull request aur GitHub push karta hoon.

# **OOP & Core Concepts**

### 5. Object-oriented programming ki 4 basic principles batao aur ek example Node.js mein do.

Encapsulation, Inheritance, Polymorphism, Abstraction. Example: User class aur Admin class backend authentication system mein.

### 6. Class aur object mein difference simple example ke saath samjhao.

Class blueprint hoti hai, object us class ka real instance hota hai. Example: Car class, Honda object.

### 7. Encapsulation ko backend projects mein kaise implement karte ho?

Sensitive data ko private methods/variables mein hide karta hoon aur controlled access APIs provide karta hoon.

### 8. Inheritance ka use kab aur kahan karna chahiye backend mein?

Jab common functionality multiple classes mein reuse karni ho. Example: Base User class se Admin/User extend karna.

### 9. Polymorphism ka ek practical example do using JavaScript ya Python.

Same method different behavior show kare. Example: payment.process() Stripe aur PayPal dono mein different implementation.

### 10. Abstraction ka matlab aur isko Node.js API design mein kaise use karte ho?

Complex implementation hide karna aur simple interface dena. Example: service layer APIs.

### 11. Constructor aur destructor ka concept explain karo.

Constructor object create hone par initialize karta hai. Destructor memory cleanup karta hai.

### 12. Static methods aur instance methods mein difference batao.

Static methods class se call hote hain, instance methods object se.

### 13. Exception handling backend mein kaise implement karte ho?

try-catch blocks, centralized error middleware aur proper status codes use karta hoon.

### 14. Middleware ko OOP design ke context mein kaise use karte ho Node.js mein?

Middleware reusable modules ki tarah use hota hai for authentication, validation, logging.

### 15. Data validation aur sanitization ke liye OOP approach kaise design karte ho?

Separate validator classes aur reusable validation methods create karta hoon.

### 16. Singleton pattern kya hai aur backend project mein iska use kahan hoga?

Single instance maintain karta hai. Example: database connection manager.

### 17. Dependency injection kya hai aur Node.js mein kaise implement karte ho?

Dependencies externally pass karna instead of hardcoding. Example: services constructor mein inject karna.

### 18. Promises aur async/await ko OOP design ke context mein kaise integrate karte ho?

Async methods classes ke andar use karta hoon for database/API operations.

### 19. REST API models aur controllers ko OOP principles ke saath kaise design karte ho?

Models data manage karte hain aur controllers business logic separate rakhte hain.

### 20. Unit testing OOP-based backend code mein kaise implement karte ho?

Jest/Mocha se individual methods aur services isolate karke test karta hoon.

# **Frameworks & Tools**

### 21. MERN stack mein MongoDB aur Node.js ke beech data flow kaise hota hai?

Frontend request bhejta hai → Express API → Mongoose → MongoDB → response frontend ko.

### 22. Express.js ya FastAPI mein routing aur middleware ka difference samjhao.

Routing endpoints define karta hai, middleware request process karta hai before response.

### 23. Laravel Sanctum authentication ka flow simple words mein batao.

User login karta hai → token generate hota hai → token har request ke saath verify hota hai.

### 24. Tailwind CSS backend ke liye kyun use kiya jaa sakta hai agar UI bhi manage karna ho?

Fast responsive UI development aur reusable utility classes provide karta hai.

### 25. Git aur GitHub workflow mein pull requests aur branching strategy kaise manage karte ho?

Main branch stable rakhta hoon, feature branches use karta hoon, then PR review ke baad merge.

### 26. Node.js mein error handling aur logging best practices kya hain?

Centralized middleware, Winston/Morgan logging aur proper HTTP status codes use karta hoon.

### 27. API performance optimization ke liye tum kya techniques use karte ho?

Caching, indexing, pagination, async processing aur query optimization.

### 28. JWT aur session-based authentication ka difference batao.

JWT stateless hota hai, session authentication server-side storage use karta hai.

### 29. Database indexing aur query optimization MERN backend mein kaise implement karte ho?

Frequently searched fields pe indexing aur unnecessary queries avoid karta hoon.

### 30. CORS problem backend mein kyun aata hai aur kaise solve karte ho?

Different domains requests block hoti hain, cors middleware se allow karta hoon.

### 31. Webhooks aur event-driven architecture ka backend mein kya role hai?

Real-time communication aur automated event processing ke liye use hota hai.

### 32. Firestore ya MariaDB mein data consistency maintain karne ka approach kya hai?

Transactions, validation aur proper schema constraints use karta hoon.

### 33. Python ke libraries (like Pandas/Requests) backend tasks mein kaise help karte hain?

Requests APIs consume karta hai aur Pandas analytics/data processing mein help karta hai.

### 34. Payment gateway integration (Stripe) secure kaise karte ho?

Webhook verification, HTTPS, tokenization aur secret keys environment variables mein store karta hoon.

### 35. API versioning ka importance kya hai?

Old clients break nahi hote aur backward compatibility maintain rehti hai.

### 36. Node.js aur Python ke asynchronous tasks compare karo in real-world backend use.

Node.js real-time apps ke liye fast hai, Python AI/data-heavy asynchronous tasks ke liye better hai.
